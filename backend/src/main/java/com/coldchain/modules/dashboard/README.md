# 指挥大屏模块（VM-001）

## 功能

- 6 项 KPI：温度达标率、设备在线率、运输中订单、预警总数、今日品质分、碳排放累计
- 图表：24h 温度趋势、告警类型分布、运输状态统计
- 实时告警滚动列表

## 接口

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/dashboard/overview` | KPI + 图表 |
| GET | `/api/dashboard/realtime-alarms?limit=10` | 滚动告警 |

## 刷新建议

前端每 15 秒拉取 overview，每 10 秒拉取告警列表。
